# Product Requirements for RoomLight

| ID  | Requirement                                                                                  | Type           | Priority |
| --- | -------------------------------------------------------------------------------------------- | -------------- | -------- |
| 001 | The lighting scheme can be designed once and synced to every room through one control system | Functional     | Must     |
| 002 | Different areas can be synced to different lighting schemes                                  | Functional     | Should   |
| 003 | Rooms are set to the default lighting automatically between guests                           | Functional     | Must     |
| 004 | The brightness and temperature of the lights change in sync with the time of day             | Functional     | Should   |
| 005 | Lighting changes can be scheduled from the visual interface                                  | Functional     | Should   |
| 006 | The system has ready-made lighting presets that the staff can apply with one tap             | Functional     | Should   |
| 007 | Guests can turn off the lights without getting up from bed                                   | Functional     | Should   |
| 008 | Bathrooms have automatic lights                                                              | Functional     | Could    |
| 009 | The brightness and temperature of the bathroom lights change in sync with the time of day    | Functional     | Could    |
| 010 | LightRoom's visual interface has different control panels for hotel staff and guests         | Functional     | Must     |
| 011 | The guest interface has simplified controls and presets relevant to the room-level lighting  | Functional     | Should   |
| 012 | The staff can see precisely which area they are changing the lighting for                    | Functional     | Must     |
| 013 | All lighting controls can be accessed by staff from a single unified view                    | Functional     | Should   |
| 014 | The visual interface should update automatically without the need for external IT support    | Non-functional | Should   |
| 015 | The system update should not disable or disrupt the lighting in the hotel                    | Non-functional | Must     |
| 016 | The light fixtures should be of a standard type                                              | Non-functional | Should   |
| 017 | The light fixtures and mounts used are compliant with the safety regulations                 | Non-functional | Must     |
| 018 | Specialised technicians should be available to fix hardware and system failures              | Non-functional | Must     |
