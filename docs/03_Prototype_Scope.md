# Prototype Scope RoomLight

### Prototype description

The RoomLight prototype demonstrates how hotel lighting can be designed once and
synced across all rooms through a single control system. It includes a unified
staff dashboard where all controls are accessible through one view, as well as a
simple guest interface that allows for room-level lighting adjustments. The
prototype includes scheduled lighting changes and area-specific configurations,
giving staff flexible control and customization options. Together, these
features demonstrate the system's ease of use, scalability and the fact it
requires no technical expertise or IT support to operate.

### Demonstrate

REQ-001 The lighting scheme can be designed once and synced to every room
through one control system.

- The prototype shows a central control system through which the hotel staff can
  adjust the lighting and sync it to all rooms with one action.

REQ-002 Different areas can be synced to different lighting schemes.

- The prototype includes area-based controls, allowing, for example, renovated
  areas to be lit with a specific lighting without affecting the lighting in
  other rooms.

REQ-005 Lighting changes can be scheduled from the visual interface.

- The prototype includes a scheduling tool that allows staff to automate
  lighting changes, reducing the need for involvement from IT and other
  technicians.

REQ-010 LightRoom's visual interface has different control panels for hotel
staff and guests.

- The prototype includes two interfaces, one for staff and the other for guests,
  that allows non-technical users to make changes to the lighting.

REQ-013 All lighting controls can be accessed by staff from a single unified
view

- The simplicity of the interface allows for non-technical staff, such as night
  managers, to adjust the lighting as needed.
